# Agent Commander OS

This is a minimal system to capture real-time demand signals and convert them into business opportunities.

## Goal

Find 20 high-intent potential customers within 7 days.

## Workflow

Day 1-2: Demand signal scanning  
Day 3: Filtering (100 → 20)  
Day 4-5: Deep understanding  
Day 6: Outreach  
Day 7: Review  

## Agents

- Demand Scanner Agent
- Signal Filter Agent
- Outreach Preparation Agent
- Commander Agent

## First Task

Scan for demand signals in:

- LinkedIn
- Google News
- Amazon / Shopify
- Reddit / Forums

Focus:

- Homecare devices
- Sleep-related products
- US / UK / Germany markets
