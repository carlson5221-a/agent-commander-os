# Agent Commander 定位文档

## 一句话定位

我站在「中国供给 × 世界需求 × Agent Commander」的交叉点，用 AI/Agent 帮助 SME 更低成本、更高精度地发现需求、建立信任、促成成交。

## 我是谁

我不是单纯的工具使用者，也不是单纯的 AI 咨询者。

我的角色更接近：

> 业务第一性思考者 + 全球需求观察者 + Agent 指挥官

## 长期母题

中国供给会长期存在。世界需求也会长期存在。真正困难的是：

> 如何在需求碎片化、渠道碎片化、信任碎片化的时代，重新提高供需匹配效率。

## Agent Commander 的工作边界

我不追逐概念，不为使用工具而使用工具。

我关注：

1. AI 是否进入真实业务流程
2. Agent 是否提升成交概率
3. 系统是否沉淀可复用经验
4. 是否产生真实商业结果

## 核心原则

> 不追 Agent，追业务闭环。  
> 不追 Skill，追成交变量。  
> 不追概念，追 AI 是否进入真实生产函数。
